Проект: Онлайн-платформа для організації групових спортивних тренувань

Опис:
Цей проєкт є веб-платформою, що дозволяє замовити та організувати групові спортивні тренування. 

Основні функції:
- Реєстрація та авторизація користувачів
- Пошук за критеріями на сайті.
- Банер з запрошенням на роботу.
- Доступний інтерфейс.
